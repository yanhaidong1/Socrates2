Prepare genes with Tn5 insertions

This step is to build a file recording the gene with Tn5 insertions. It has three columns: 1) gene ID 2) cell barcode 3) Tn5 insertion count.

Inputs (Required): 1. -open_geneTn5 yes 2. -script_dir (a script directory downloaded from GitHub: Socrates2/utils/input_other_required_scripts_dir) 3. -tn5_fl (A file obtained from Process BAM files obtained from cellranger-atac tool step which records the coordinates of each Tn5 insertions per cell) 4. -Ggff_fl (A gene gff annotation file) 5. -Genome_fl (A genome fasta file) 6. -Gmfai_fl (A genome index file, prepared by ‘samtools faidx’) 7. -o output_dir (mkdir output_dir)

Optional parameters 1. -category (Set gene or transcript to be calculated. Default: gene.)

Outputs: 1. opt_genome_size.txt 2. opt_gene_tn5.sparse 3. opt_genes_500bpTSS_sorted.bed

Running: python Socrates2/utils/prepare_tn5_gene_accessibiliy.py
-script_dir Socrates2/utils/input_other_required_scripts_dir
-o output_dir
-tn5_fl /Path/to/opt_tn5_mq10.bed
-Ggff_fl /Path/to/Gene.gff3
-Genome_fl /Path/to/Genome.fasta
-Gmfai_fl /Path/to/Genome.fasta.fai

Calculate gene accessibility

This step is to calculate the gene accessibility. The output file has three columns: 1) gene ID 2) cell barcode 3) Accessibility score.

Inputs (Required): 1. -open_geneAcc yes 2. -script_dir (a script directory downloaded from GitHub: Socrates2/utils/input_other_required_scripts_dir) 3. -meta_fl (A meta file obtained from Loading pre-processed data, dimensionality reduction, and clustering) 4. -gene_tn5_fl (An opt_gene_tn5.sparse file obtained from Prepare genes with Tn5 insertions step) 5. -gene_bed_fl (An opt_genes_500bpTSS_sorted file obtained from Prepare genes with Tn5 insertions step) 6. -o output_dir (mkdir output_dir)

Outputs: 1. opt_gene_body_accessibility_mtx.rds

Running: python Socrates2/utils/prepare_tn5_gene_accessibiliy.py
-script_dir Socrates2/utils/input_other_required_scripts_dir
-o output_dir
-meta_fl /Path/to/meta.txt
-gene_tn5_fl /Path/to/opt_gene_tn5.sparse
-gene_bed_fl /Path/to/opt_genes_500bpTSS_sorted

Smooth gene accessibility

The smooth step will smooth gene accessibility using a published method ‘MAGIC’ referred from David van Dijk, et al. Recovering Gene Interactions from Single-Cell Data Using Data Diffusion. 2018. Cell.

Inputs (Required): 1. -open_smoothGeneAcc yes 2. -script_dir (a script directory downloaded from GitHub: Socrates2/utils/input_other_required_scripts_dir) 3. -meta_fl (A meta file obtained from Loading pre-processed data, dimensionality reduction, and clustering) 4. -svd_fl (A svd file obtained from Loading pre-processed data, dimensionality reduction, and clustering) 5. -gene_acc_fl (An opt_gene_body_accessibility_mtx.rds file obtained from Calculate gene accessibility step) 6. -o output_dir (mkdir output_dir)

Optional parameters: 1. -core (Set cores we will use for the parallele analysis. Default: 1.) 2. -clustnm (Set the cell cluster name within meta file. Default: LouvainClusters.)

Outputs: 1. opt_allgenes_impute.activity.rds

Running: python Socrates2/utils/prepare_tn5_gene_accessibiliy.py
-script_dir Socrates2/utils/input_other_required_scripts_dir
-o output_dir
-open_smoothGeneAcc yes
-meta_fl /Path/to/meta.txt
-svd_fl /Path/to/svd.txt
-gene_acc_fl /Path/to/opt_gene_body_accessibility_mtx.rds
